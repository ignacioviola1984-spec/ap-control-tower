# AP Control Tower

Sistema de agentes maker-checker para el proceso de Cuentas a Pagar (AP/P2P) de una consultora.
Demo comercial: corre 100% local con datos sinteticos, sin API keys, sin integraciones externas, sin acceso a red.

> **El sistema bloquea solo. Liberar dinero siempre es humano.**

Repositorio privado. El material de referencia del proceso real vive en `docs/` (gitignoreado, nunca se commitea).
